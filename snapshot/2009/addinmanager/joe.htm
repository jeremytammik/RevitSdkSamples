Hi Joe,

 

The install error is caused by your Revit Version. Currently the AddInManager Installer is just work on the release version Revit.  Your Revit is preview beta version, so you can’t install it successfully. There is a method to fix this. You can modify some registry information of Revit products by the regedit Tool.  Modify the Display name of your revit product in the HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\Uninstall. For example, change “Revit Architecture 2009 C Preview Beta” to “Revit Architecture 2009”.

 

I have checked the bug that it can’t save all the commands in one assembly. It’s true about that; I will fix this in next version of AddInManager.
